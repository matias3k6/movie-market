# Hello There!

This is App let you search for a movie and get some details.

## Run the App

### Clone the project

```
git clone https://github.com/matias3k6/movie-market.git
```
or
```
git clone git@github.com:matias3k6/movie-market.git
```

### Install dependencies

```
yarn install
```

or

```
npm install
```

### Start

```
yarn start
```

or

```
npm start
```
